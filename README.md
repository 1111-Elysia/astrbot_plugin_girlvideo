# astrbot_plugin_girlvideo

B站视频搜索推送插件 — 输入关键词自动搜索B站视频并下载发送到群聊。

## 功能

- **关键词触发**：消息包含配置的触发词时自动搜索B站
- **定时推送**：可配置多个定时任务，按间隔自动搜索并发送到指定群聊
- **去重过滤**：已发送的视频不会重复推送
- **减词回退**：多关键词无结果时自动减少关键词重试
- **逐个尝试**：下载失败自动换下一个候选视频
- **大小限制**：超过配置大小的视频自动跳过
- **媒体中转**：文件转为公网URL，解决协议端跨容器访问问题

## 依赖

- `aiohttp`
- `astrbot_plugin_media_parser` (v6.3.1+)

## 配置

### 触发设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `trigger.keywords` | list | `["搜视频"]` | 触发关键词，消息包含任意一个即触发 |

### 中转设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `relay.enable` | bool | `true` | 启用媒体中转 |
| `relay.callback_url` | string | `""` | AstrBot 公网地址，如 `http://your-server:6185` |
| `relay.ttl` | int | `300` | 文件 token 有效期(秒) |

### B站设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `bilibili.cookie` | string | `""` | B站 Cookie（含 SESSDATA），提升搜索稳定性 |

### 定时任务

可添加多个定时任务，每个任务独立配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enabled` | bool | `true` | 启用此任务 |
| `keyword` | string | `""` | 搜索关键词 |
| `interval_minutes` | int | `60` | 搜索间隔（分钟） |
| `count` | int | `1` | 每次发送视频数量 |
| `order` | string | `"综合排序"` | 排序方式：综合排序/最多播放/最新发布/最多弹幕 |
| `max_size_mb` | int | `50` | 视频大小上限(MB)，0=不限制 |
| `target_groups` | list | `[]` | 目标群聊ID列表 |

## 使用

### 手动触发

在群聊中发送：

```
搜视频 可爱猫咪
搜视频 搞笑视频 --数量 3
```

支持空格分隔的多关键词（减词回退）：

```
搜视频 女神 性感 宅舞
```

### 定时任务

在 WebUI 插件配置中添加定时任务，插件会自动按间隔搜索并发送到目标群聊。重载后自动恢复 origin 缓存，无需手动操作。
